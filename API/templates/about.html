About this Site

Hey! This is Sriram. Welcome to my demo Site. Objective is to showcase 3 business use cases, I developed and deployed ML models.

Image Classification

This is a typical use case in Healthcare Paper Claim processing workflow. First step in the process is to classify the incoming images. This critical step helps with completing the KFI, in order for the claim to be properly filed for payment. Image classification API building using CNN model helps to identify Structured and Unstructed Claims, thus trying to reduce / eliminate manual classification effort.

API can be accessed using /Image for prediction

Enrollment Status Review

This use case involves assisting Eligibility Underwriters make determination either to approve or reject enrollment application received during Special Enrollment Period. Prediction are made using Member and Dependent Demographics through Logistic Regression Model.

API can be accessed using /Enrollmentreview for prediction

Utilization Review

High Dollar Claims go through utilization approval prior to payment. This involves capturing additional Meta data from Itemized Bill. API built using LSTM helps identify claims that require additional keying using Medical report finding, thus enabling only qualified claims to be routed for additional review.

API can be accessed using /Utilizationreview for prediction
